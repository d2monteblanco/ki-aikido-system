Verifique se digitou o email corretamente 2. Verifique a senha (maiúsculas/minúsculas) 3. Tente limpar o cache do navegador (Ctrl+Shift+Del) 4. Entre em contato com administrador ``` ### ? "Não encontro um aluno" ``` 1. Use a barra de busca (topo da tabela) 2. Digite parte do nome ou CPF 3. Verifique se está na aba correta (Alunos vs Membros) 4. Rolagem: pode estar em outra página ``` ### ? "Não consigo adicionar certificado" ``` 1. Verifique o tamanho (máximo 5 MB) 2. Verifique o formato (PDF, JPG, PNG) 3. Tente reduzir o tamanho do arquivo 4. Tente outro formato (ex: PDF em vez de JPG) ``` ### ? "Adicionei foto mas não aparece" ``` 1. Aguarde alguns segundos (upload demora) 2. Atualize a página (F5) 3. Verifique se salvou o formulário 4. Tente adicionar novamente ``` ### ? "Marcado 'Graduação Atual' mas ainda aparece outra" ``` 1. Edite a graduação antiga 2. Desmarque "Graduação Atual" 3. Salve 4. Atualize a página ``` --- ## ?? Atalhos do Teclado ### Navegação ``` Tab ? Próximo campo do formulário Shift+Tab ? Campo anterior Enter ?
